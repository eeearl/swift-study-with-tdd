# Swift study with TDD
TDD를 적용한 스위프트 스터디 기록용 저장소.

## Documentation Tool
[Jingo](https://github.com/claudioc/jingo)를 사용해 작성과 관리함

### Installation & Setting
```
sudo npm install -g jingo
jingo -s > config.yaml
jingo -# [password]
```
config.yaml 파일에서 `repository`, `local` 을 필수로 채워넣는다.

### Run
```
jingo -c config.yaml
```
